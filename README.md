# neotimer
webpage for a sleek timer and to-do App (no backend). 
**Live Demo** View [here.]{neotimer.jaegerlab.link}

# description
Start your pomodoro timers as you like. 25/5 50/10 or however you want to configure them. <br>
Save your to-dos on a sticky-note and check them off as long as you remain in the browser tab
